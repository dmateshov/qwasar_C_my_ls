# qwasar_C_my_ls
rework of my_ls function in C
# Qwasar Silicon Valley
## my_printf project
 
This project is torework of "ls" Unix command that lists files in C. 
 

### Compiling

Run the following commands:

* To compile
	- `make`
* To remove objects:
	- `make clean`
* To remove objects and binary file (program):
	- `make fclean`
* To re-compile:
	- `make re`
